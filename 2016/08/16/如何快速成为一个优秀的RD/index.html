(逐字稿)如何快速成为一个优秀的RD

发表于 2016-08-16   |   分类于 逐字稿   |     |   次阅读

最近要给新入职的RD同学们分享一下如何快速成长，大致整理了一下思路，在这里记录一下以作备忘。

如何快速成长成一名优秀的RD。

大家好，我是刘新宇，目前也就入职1年多，就以我个人经验给大家回答一下这个问题，如何快速成长虽然没有捷径可走，但是也有很多技巧的，在这里就给大家分享4点比较实用的。

第一点：工作中做事一定要有目标、有计划、有总结

不管是学习一项技能还是做一个业务需求，一定要在初期开始做的时候就要有一个明确的目标，并有可执行的计划，初期的话可以跟自己导师请教，同时在最后做完之后也要有总结，总结最好是落实到书面上的，类似日报、周报这种形式。

第二点：要保持积极学习的态度

大家都知道，我们互联网行业的技术革新日新月异，如果不积极学习很快就会掉队。学习方法的话就不多讲了，在这里给大家推荐两个学习途径：

  • 读书平台。美团图书馆，6600多本同事共享的书,技术上的，非技术上的，利用上下班的班车上，地铁上的空闲时间读书，能学到不少东西。

  • 分享平台。在新美大我们有一个分享平台，不知道大家知不知道，上面有很多优质的资源，可以面对面沟通，多听前辈们的分享，能让你在工作中少走很多弯路。

第三点：业务与技术一定要相辅相成

二者不能丢掉任一方，任何技术都是为业务服务，业务也是用来实践技术的，做业务并不是一味的重复劳动，做的时候一定要有自己的思考，把自己会的技术尽可能的应用进去，千万不要PM让做什么就做什么，把平凡的事情做出不平凡来。

第四点：有问题的地方就有机会

作为新人不要胆怯，觉得工作中有哪些不合理的地方要立刻指出来，只要说得对，大家都会听你的。一定要把握机会让大家认可你。

归根结底，如果说相比别人优秀那肯定就要比别人多一些付出，祝大家早日成为独当一面的大牛。
